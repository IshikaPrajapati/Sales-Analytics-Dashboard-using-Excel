# Sales Analytics Dashboard using Excel

#### Project Overview



An interactive sales analytics dashboard built using Excel Pivot Tables, Pivot Charts, Slicers, and KPI Cards.



##### Features:

* Total Revenue Analysis
* Average Order Value (AOV)
* Active Customer Tracking
* Sales Trend Analysis
* Product Line Performance
* Country-wise Sales Analysis
* Interactive Slicers



##### Tools Used:

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting



##### Key Insights:

* Classic Cars are the strongest revenue driver.
* Medium-sized deals dominate sales.
* Australia and USA contribute significant revenue.
* Repeat customers generate a large share of sales.
* Sales peak during year-end periods.
